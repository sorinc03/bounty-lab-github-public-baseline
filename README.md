# GitHub public baseline bounty lab

Owned non-sensitive test repo for GitHub Bug Bounty baseline research.

Created: 2026-05-20
Owner: `sorinc03`
Remote: https://github.com/sorinc03/bounty-lab-github-public-baseline
Visibility: public

## Purpose

This repository is intentionally public and contains no secrets, private customer data, or sensitive business data. It exists to run free, low-frequency GitHub Actions baseline tests before any private-repo boundary testing.

## Rules

- Keep all data harmless and public.
- Do not add secrets.
- Use manual workflow runs only.
- Do not run load tests, spam, scanners, or third-party callbacks.
- Record every experiment before and after running it.
